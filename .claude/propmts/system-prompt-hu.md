Egy rövid, maghyar nyelvű online kurzust állítunk össze a MITS Pilothoz (`/.claude/context/mits-pilot-md`), ami egy modern frontend alkalmazás iparági legjobb gyakorlatnak megfelelő elkészítésébe vezeti be a "REST API Frontend Development" témakörének feladatmegoldására készülő diákokat.
A kurzus teljesen gyakorlatorientált, vagyis hosszas elméleti bevezető helyett konkrét gyakorlati lépésenként egy nagyon egyszerű porjekt lépésekre bontott létrehozásán keresztül mutatja be a folyamatot.
A projekt egy React TS projekt lesz, a UI kialakításához Shadcn-t és Tailwind-et használunk.
A kurzus 5-10 modulban (egy lépés egy modul) mutatja be a projekt elkészítésének folyamatát.
Minden modult külön mappában tárolunk (module-1, module-2 stb). Minden modulhoz tartozik egy rövid áttekintés (overview.md) és egy gyakorlati feladat, ahol a projekt megvalósításának aktuális lépését mutatja be egymás utáni taszkokra bontva (workshop.md). A modul mappán belül a solution mappában tároljuk a workshop.md megoldását.
A következő modul workshop kiindulóppontja mindig az előző modul vágállpota lesz.
Az overview.md, a workshop.md és a solution elkészítéséhez használd a Context7, a Tailwind és a Shadcn MCP szervereket.
