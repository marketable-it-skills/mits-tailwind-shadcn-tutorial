# MITS Tailwind CSS & ShadCN UI Tutorial

**A [Marketable IT Skills (MITS)](https://github.com/marketable-it-skills) kezdeményezés támogatásával**

Egy praktikus, lépésről lépésre tutorialt a **Tailwind CSS** és **ShadCN UI** tanulásához egy modern projekt dashboard építésével. Ez a tutorial a **MITS (Marketable IT Skills)** Pilot program része.

![MITS Tailwind CSS & ShadCN UI Tutorial](/assets/images/course-cover-image.webp)

## Mit fogsz építeni

Egy responsive MITS Project Dashboard projektkártyákkal, amely tartalmazza:

- Több színtémát (világos, sötét, színes akcentusok)
- Professzionális tipográfiai hierarchiát
- ShadCN UI komponenseket (Card, Badge)
- Verseny badgeket és technológia tageket
- Interaktív hover effekteket és animációkat
- Akadálymentes vezérlést billentyűzettel (WCAG 2.1 megfelelő)
- Responsive grid layoutot
- Dark/light mode váltót témamentéssel

![MITS Tailwind CSS & ShadCN UI Tutorial](/assets/images/course-solution.webp)

## Tech Stack

- **Vite** - Gyors build eszköz és fejlesztői szerver
- **React 18** - Felhasználói felület könyvtár
- **TypeScript** - Típusbiztos JavaScript
- **Tailwind CSS v4** - Utility-alapú CSS keretrendszer
- **ShadCN UI** - Akadálymentes komponens könyvtár (Module 4+)

## Tanulási Filozófia

Ez a tutorial egy **UI-first, visual-first megközelítést** követ:

1. **Azonnali eredmények** - Építs vizuális komponenseket a kezdetektől
2. **Cselekvéssel tanulás** - Praktikus gyakorlatok, nem csak olvasás
3. **Progresszív komplexitás** - Kezdj egyszerűen, adj hozzá funkciókat fokozatosan
4. **Értsd meg mielőtt absztrakciókat használsz** - Tanuld meg a Tailwind alapokat, mielőtt komponens könyvtárakat használnál
5. **Professzionális gyakorlatok** - Git workflow, verziókezelés, best practicek

## Kezdés

### Előfeltételek

- **Node.js** v18+ ([Letöltés](https://nodejs.org/))
- **Git** ([Letöltés](https://git-scm.com/))
- **GitHub account** ([Regisztráció](https://github.com))
- Alapvető HTML, CSS és JavaScript ismeretek
- React alapok ismerete

### Gyors Kezdés

1. **Klónozd a repositoryt:**

   ```bash
   git clone https://github.com/marketable-it-skills/mits-tailwind-shadcn-tutorial.git
   cd mits-tailwind-shadcn-tutorial
   ```

2. **Kezdd a Module 1-gyel:**
   - Olvasd el a [`module-1/overview.md`](./module-1/overview.md) fájlt
   - Kövesd a [`module-1/workshop.md`](./module-1/workshop.md) útmutatást
   - Hasonlítsd össze a megoldásodat a `module-1/solution/` mappával

3. **Folytasd progresszíven a modulokkal** (Mind a 10 modul kész!)

## Hogyan használd ezt a tutorialt

Minden modul tartalmazza:

```
module-X/
├── overview.md          # Tanulási célok és kontextus
├── workshop.md          # Lépésről lépésre praktikus gyakorlatok
└── solution/            # Teljes működő megoldás
    ├── README.md        # Megoldás dokumentáció
    └── src/             # Source code
```

### Ajánlott Megközelítés

1. **Olvasd el az overviewt** - Értsd meg, mit fogsz tanulni és miért
2. **Kövesd a workshopt** - Végezd el minden feladatot lépésről lépésre
3. **Kísérletezz** - Próbáld ki a challenge gyakorlatokat
4. **Hasonlítsd össze a megoldással** - Ellenőrizd a munkádat a biztosított megoldással
5. **Commitold a munkádat** - Használd a Git-et a haladásod követéséhez
